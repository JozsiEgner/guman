# Guman

Guman is the foundational runtime for the Ézótóo project.

## Core idea

A human-patterned spatial entity system with:
- male and female base body archetypes
- an internal Ézó core
- controlled self-division logic
- authorization-gated replication
- dynamic state-driven behavior

## First foundation included

- domain model for body, core, state and permissions
- controlled division policy
- minimal runtime example
- TypeScript project structure

## Planned next layers

1. 3D body schema
2. motion and state transitions
3. Ézó energy channels
4. replication inheritance rules
5. visual simulation runtime

## Start

```bash
npm install
npm run dev
```
